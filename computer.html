BCA (Cloud Computing, Cybersecurity and Ethical Hacking)

3-Year Undergraduate Program

A comprehensive program focusing on cloud technologies, cybersecurity fundamentals, and ethical hacking practices. Students learn to deploy cloud solutions while ensuring security best practices.

Program Structure

  • Cloud Computing Fundamentals
  • Network Security
  • Ethical Hacking Methodologies
  • Programming Languages (Python, Java)
  • Database Management Systems
  • Web Technologies
  • Operating Systems
  • Cloud Platforms (AWS, Azure)

Eligibility Criteria

  • 10+2 with minimum 50% marks
  • Mathematics as compulsory subject
  • Qualifying score in entrance exams (SET, CUET, IPM)
  • Basic computer knowledge

Career Opportunities

  • Cloud Security Specialist
  • Ethical Hacker
  • Security Analyst
  • Cloud Administrator
  • Network Security Engineer
